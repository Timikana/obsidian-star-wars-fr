# Mandaloriens
> [!color] Allégeance
> <span style="color:#4FC3F7">█</span> **Bleu Mandalorien** — Peuple guerrier de [[Mandalore]] uni par un code et le crâne de Mythosaure.

## Résumé
Culture guerrière millénaire forgée dans le combat, les Mandaloriens se divisent en factions rivales partageant une armure de beskar et le code "This is the Way". Décimés par la Grande Purge impériale, ils sont reconquis par [[Bo-Katan Kryze]] et [[Din Djarin]].

## Hiérarchie / Membres clés
- **Death Watch** — [[Pré Vizsla]]
- **True Mandalorians**
- **Children of the Watch** — [[Din Djarin]], [[L'Armurière]]
- **Nite Owls** — [[Bo-Katan Kryze]]

## Symboles & Code
- Crâne de Mythosaure
- Sabre Noir (Darksaber)
- "This is the Way"

## Apparitions
The Clone Wars, Rebels, The Mandalorian, The Book of Boba Fett, Ahsoka.
