## 🎨 Activation des couleurs

1. **`Ctrl+R`** dans Obsidian (recharger)
2. Settings → **Apparence → Snippets CSS** → activer `star-wars-colors`
3. **`Ctrl+G`** pour ouvrir le graphe coloré (35 groupes de couleurs)

Le graphe colore automatiquement chaque nœud selon :
- **Personnages** : couleur de sabre laser (priorité 1) ou faction (priorité 2)
- **Planètes** : région galactique (Core → Unknown Regions)
- **Vaisseaux** : faction (Empire / Rébellion / République / CIS / Premier Ordre)
- **Créatures** : type (fauve / monture / aquatique / mythique)
- **Organisations** : allégeance
