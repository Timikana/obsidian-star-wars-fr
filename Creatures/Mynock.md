# Mynock
> [!color] Type
> <span style="color:#6D4C41">█</span> **Fauve** — parasite stellaire ailé qui se nourrit d'énergie électrique et de gaz.

## Résumé
Les mynocks vivent dans le vide, accrochés aux astéroïdes ou aux coques des vaisseaux dont ils drainent les câbles. Capables de survivre dans le ventre d'un [[Exogorth]], ils infestent les grottes spatiales et les épaves abandonnées.
